# 采集域稳健的酸枣仁高光谱产地溯源：方法冻结与一次性锁定确认方案

> 文档状态：结果前（result-before-freeze）预注册方案
> 建立日期：2026-07-22
> 关联记录：R0031、R0032、R0038、R0039、R0040、R0041
> 适用范围：`main` 分支产地溯源方法的第二轮冻结与锁定确认
> 证据边界：本方案在读取任何预留批次（构造批次 8–9）数值之前写定。所有开发数值只来自构造批次 0–7；构造批次 8–9 的数值与文件字节读取当前为 0。

## 1. 目标与科研定位

唯一科研目标不变：利用 392 波段近红外高光谱对酸枣仁进行八产地地理溯源。本方案针对该目标下**当前数据能够诚实回答的最重要问题**：

> 当采集立方体（来源高光谱图像）发生变化时，产地分类是否仍然可靠且校准良好？

同域（同一采集立方体内）随机或留一构造批次评估已接近饱和（各模型平衡准确率约 97%–98%，模型间差异仅 1–2 粒），不能区分方法优劣，也不能反映真实部署时的采集域变化。因此本方案将**整幅来源立方体迁移（whole-source-cube transfer）**确立为主评估轴。

本方案的贡献分三部分，均为诚实、有边界的贡献，不宣称全新神经网络架构：

1. **方法与协议贡献**：建立防泄漏的整幅立方体跨采集域产地溯源评价协议，避免同一立方体内随机划分造成的虚高表现。
2. **实证贡献**：证明 SG 一阶导数 + 收缩 LDA 比 SNV、MSC、二阶导数、逻辑回归、RBF-SVM 及深度对照具有更好的跨采集稳定性；批次级无监督目标采集标准化能进一步降低误差；温度缩放修复跨采集置信度失真。
3. **机制与负面结果贡献**：证明逐种子像素协方差与种子内部纹理分布在当前数据中主要携带采集特异性信息而非稳定产地信息；并证明分组（shift-aware）温度校准与分组保形集相对普通 iid 版本没有提供预期优势。

## 2. 分析单位与防泄漏边界

- 分析对象：1,264 粒种子，8 产地，每产地 2 张来源图像（立方体 `*-1`、`*-2`）。
- 构造批次：沿用 R0032 的确定性分配（子种子 `20260721 + label*101 + replicate*1009`，排序后置换秩对 10 取模）。批次 0–7 为开发（1,012 粒），批次 8–9 为一次性锁定确认预留（252 粒）。
- 采集域轴：立方体后缀（`replicate ∈ {1,2}`）是当前可恢复的唯一采集边界。整幅立方体迁移即在两张来源图像之间训练/测试。
- 开发阶段只用 `discover_manifest(hash_files=False)` 枚举结构，只加载 development 划分；任何锁定记录在首次 I/O 前被硬拒绝。

## 3. 冻结流水线（部署主预测器）

冻结实现：[`provenance_study/acquisition_robust_pipeline.py`](../provenance_study/acquisition_robust_pipeline.py) 中的 `AcquisitionRobustProvenanceClassifier`。逐步骤为：

1. **表示**：Savitzky–Golay 一阶导数（窗口 15、二阶多项式、`mode=interp`）。
2. **训练标准化**：在训练立方体上拟合逐波段均值/标准差。
3. **无监督目标采集标准化**：进样立方体（一整批未知产地的种子）在**其自身**逐波段统计上标准化，去除逐立方体仿射批次效应，不使用任何标签。要求进样批量 ≥ 8 粒；不支持单粒评分。
4. **分类器**：收缩 LDA（`solver=lsqr`、`shrinkage=auto`），在训练立方体自标准化特征上拟合。
5. **温度校准**：单一标量温度 `T`，在训练立方体的构造批次分组 OOF（每次留出一个构造批次、留出批以其自身统计标准化）上以多分类 NLL 最小化拟合。

冻结常量：`SG_WINDOW=15, SG_POLYORDER=2, SG_DERIVATIVE=1, LDA_SOLVER=lsqr, LDA_SHRINKAGE=auto, TEMPERATURE_GRID=exp(linspace(log0.25, log8, 400)), MIN_INCOMING_BATCH=8`。

## 5. 预注册一次性锁定确认设计

锁定确认的唯一目的：检验第 3 节冻结流水线与第 4 节开发效应在**预留批次 8–9**上是否复现。设计固定如下，读取锁定数据前不得更改：

- 方向 A：训练 = 立方体 `*-1` 的构造批次 0–7；测试 = 立方体 `*-2` 的构造批次 **8–9**。
- 方向 B：训练 = 立方体 `*-2` 的构造批次 0–7；测试 = 立方体 `*-1` 的构造批次 **8–9**。
- 测试种子既属于预留批次（8–9），又来自相反采集立方体，因此是同时跨采集域且从未参与开发选择的一次性held-out。
- 主指标 θ = 两方向平衡准确率等权平均；次要指标 = 准确率、macro-F1、NLL、ECE、Brier、逐产地召回、混淆矩阵。
- 不确定性：以产地×测试批次为簇的 2,000 次分组自助 95% 区间；两方向分别报告并合并。
- 执行守卫：确认短语 `UNLOCK_BATCHES_8_9`、干净 Git 工作树、无既有 `executed_complete` 状态；完成后写不可覆盖状态。锁定读取只允许发生一次。

### 5.1 预注册效应门槛（方向性复现，非事后阈值）

冻结流水线被判定“采集域稳健性在预留数据上复现”当且仅当以下全部成立：

1. 冻结流水线锁定 θ 明显高于 SNV-LR 弱基线（点差 > 0 且簇自助下界 > 0）。
2. 无监督目标采集标准化在锁定测试上的平衡准确率增益 ≥ 0（两方向同号）。
3. 温度校准在锁定测试上降低 ECE 与 NLL（两方向同号）。
4. 逐种子协方差拼接在锁定测试上不提升（方向性阴性复现）。

任一条不成立时，如实报告为部分复现或阴性，不得事后放宽门槛或更换流水线。

## 6. 强基线、对照与保留的阴性结果

- 强/弱光谱基线：raw、SNV、MSC、SG1、SG2 × LDA/LR/RBF-SVM/等权集成，均在同一整幅立方体轴报告（`cross_cube_metrics.csv`）。
- 深度对照：R0021 记录的谱空融合网络整幅立方体 θ≈45.9%，SNV-LR≈86.9%，作为“复杂空间模型在采集域迁移下更脆弱”的既有证据。
- 保留阴性结果（不得隐藏）：
  - 分组（shift-aware）温度与 iid 温度不可区分（`iid_minus_shift_aware` ECE 差 −0.0007，区间跨 0）。
  - 分组与 iid 保形集在跨域下均欠覆盖（约 0.80 vs 目标 0.90），二者无实质差异。
  - 逐种子 PCA-16 协方差描述子拼接使跨立方体平衡准确率 0.9365 → 0.9204、NLL 0.239 → 0.606。

## 7. 结论边界

构造批次由导师授权、按确定性规则划分，是当前数据内部的分组单位，但仍共享每产地两张来源图像，不是新采集的物理批次。整幅立方体迁移是采集域稳健性压力测试，每产地每方向仅 1 张测试图像，不能估计跨农场、跨年份、跨仪器或未知产地的外部泛化。冻结流水线适用于当前存档闭集内的采集域稳健方法比较，不构成法律或供应链意义上的地理认证。要获得外部认证证据，仍需按 [数据采集协作需求书](./数据采集需求.md) 建立可追溯多域队列。
