# 现有数据产地溯源算法探索与锁定验证方案

## 1. 唯一研究目标

本研究的唯一任务是：**利用高光谱数据对酸枣仁进行八产地溯源**。所有数据划分、预处理、模型选择、集成、校准、消融、伪证和统计分析均必须直接服务于该目标。来源图像迁移仅是评估采集域敏感性的次要压力检验，不得替代产地溯源主分析。

## 2. 当前数据与结论边界

当前数据包含 8 个产地、1,264 粒酸枣仁、392 个波段（949.764–1650.855 nm）：HBS 160 粒、HBX 160 粒、HNA 160 粒、HNX 160 粒、NX 170 粒、SXD 160 粒、SXQ 160 粒、XJH 134 粒。每个产地由两张来源图像提取逐粒平均光谱和 MAT 补丁。

经研究团队与导师确认，允许在同一产地内将现有种子确定性划分为多个构造批次，并将构造批次作为本轮算法研究的主要内部分组单位。但是，这些批次由同一批现有图像中的种子构造，不能等同于新农场、新收获批次、新年份或外部设备数据。本研究能回答“现有数据内部的八产地判别”，不能单凭本结果宣称已完成跨年份、跨设备或未知批次的外部认证。

## 3. 构造批次和结果前锁定

批次划分完全由样本编号和固定随机种子决定，不使用光谱数值、产地可分性或模型结果：

1. 对每个 `label × source_replicate` 组合，按逐粒文件名的整数编号升序排列；
2. 使用 NumPy `default_rng(20260721 + label × 101 + source_replicate × 1009)` 生成固定置换；
3. 按置换后从 0 开始的次序取模 10，得到构造批次 0–9；
4. 两张来源图像中批次索引相同的种子共同组成该产地的一个构造批次，使每个批次都包含两张图像的样本；
5. 批次 0–7 为开发集，共 1,012 粒；批次 8–9 为程序锁定的内部测试集，共 252 粒。

开发阶段只允许枚举锁定文件的路径以建立分配表，不允许打开批次 8–9 的 CSV 或 MAT 数值。正式代码、配置、主模型、融合权重、温度校准规则、指标和效应判定门槛必须先提交 Git，然后才能进行一次锁定测试。该测试是本次算法重构的结果后锁定内部评估，但由于历史脚本曾处理全部数据，不将其冒充为从未触及的前瞻性外部数据。

## 4. 开发集算法探索

所有开发结果均由留一构造批次的 8 折 out-of-fold（OOF）预测汇总。任何 StandardScaler、MSC 参考谱、特征排名、融合权重和概率校准只能在当前训练折中拟合。开发阶段实际比较了：

- raw、SNV、MSC 和 Savitzky–Golay（SG）一阶导数光谱；
- PLS-DA、Logistic Regression（LR）、收缩线性判别分析（LDA）、RBF-SVM 和 ExtraTrees；
- SG 一阶导数的多窗口候选、非负凸权重和 top-k 集成；
- 基于前景像素标准差、四分位距、内外径向光谱差和掩膜形态的 MAT 多视图；
- 折内 ANOVA 或 LDA 系数排名的稳定选带。

开发集的主要已执行结果为：

| 开发集候选 | 平衡准确率 | macro-F1 | 审计结论 |
|---|---:|---:|---|
| SG1（15/2）+收缩 LDA | 97.5816% | 97.5850% | 最强单模型基线 |
| SG1-LDA/LR/分组温度 RBF-SVM 等权集成 | 98.2413% | 98.2396% | 净减少 7 个错误；暂定为主预测器 |
| 嵌套非负凸权重集成 | 97.7769% | 97.7814% | 权重不稳定，不升级为主模型 |
| SG1 + 全部 MAT 视图 + LDA | 97.9665% | 97.9671% | 未超过光谱集成 |
| 最佳稳定选带单模型 | 96.7027% | — | 低于全光谱 LDA，仅作负结果消融 |
| 321,776 参数的三阶段 1D 残差 CNN | 89.5597% | 89.3664% | 严格内层早停后显著低于经典光谱模型；停止扩大搜索 |

等权集成相对收缩 LDA 提高 0.6598 个百分点，平衡错误率相对减少 27.28%；它在 8 个开发折中 5 折改善、1 折下降、2 折不变，在 8 个产地中 5 类改善、2 类下降、1 类不变。这是有前景但尚不足以单独证明稳定优势的开发证据，最终判断必须依赖未参与选择的锁定批次。

## 5. 冻结主预测器

主预测器记为 `batch_constrained_sg1_probability_ensemble`，是一个描述性方法名称，不声称 SG、LDA、LR、SVM 或概率平均本身是首创算法。

### 5.1 共同预处理

每条逐粒平均光谱执行 SG 一阶导数：窗口长度 15、二阶多项式、`deriv=1`、`mode=interp`。SG 为单样本确定性变换；各模型的 StandardScaler 只在训练部分拟合。

### 5.2 三个互补成员

1. 收缩 LDA：`solver=lsqr, shrinkage=auto`；
2. 多类 LR：`C=1, solver=lbfgs, max_iter=5000`；
3. RBF-SVM：`C=10, gamma=scale, probability=False, decision_function_shape=ovr`。

SVM 不使用 libsvm 的内部非分组 Platt 校准。在全开发集内做 8 折留一批次 OOF，收集每粒的 8 类 `decision_function` 分数 (s_{ik})，然后在 (z=\log T\in[-4,4]) 内最小化多类对数损失：

\[
p_{ik}(T)=\frac{\exp(s_{ik}/T)}{\sum_j\exp(s_{ij}/T)}.
\]

正式实现以 `xatol=1e-10` 拟合得到的冻结温度为 `T=0.391012468427`（与临时探索使用默认收敛容差得到的 `0.3910119167` 仅相差约 `5.5×10⁻⁷`，不改变分类或任何报告指标）。随后使用全部 1,012 粒开发数据重新拟合三个成员，锁定测试的最终概率固定为：

\[
p_{\mathrm{ens}}=\frac{p_{\mathrm{LDA}}+p_{\mathrm{LR}}+p_{\mathrm{SVM},T}}{3}.
\]

不允许根据锁定结果删除成员、重估温度、调整权重、变更波段或更换主指标。

## 6. 锁定测试与统计分析

主终点为 16 个“产地 × 锁定构造批次”的逐批次正确率等权平均。次要终点包括逐粒 balanced accuracy、macro-F1、accuracy、多类 NLL、多类 Brier 分数、10 箱 ECE、逐产地召回率和混淆矩阵。

主方法与最强单模型 SG1-收缩 LDA 在完全相同的锁定样本上成对比较：

- 在每个产地内对两个锁定构造批次分层重采样 10,000 次，估计主指标及成对差值的百分位 95% 区间；
- 以 16 个逐批次正确率差为成对单位，计算精确 sign-flip 敏感性检验；
- 同时报告绝对百分点差、平衡错误率的相对减少及“纠正/新增”错误数。

由于每产地只有两个锁定构造批次，区间仍是对当前构造设计的条件性不确定性描述，不估计未来农场或年份的总体变异。

## 7. 结果前效应判定门槛

只有同时满足下列条件，才将主方法表述为“对强基线存在稳定且具实质性的改进”：

1. 主指标成对差值的构造批次分层 bootstrap 95% 区间下界大于 0；
2. 平衡错误率相对减少不低于 20%；
3. NLL 和 Brier 分数相对收缩 LDA 的恶化均不超过 0.01；
4. 改善不是由单一产地或单一构造批次独立驱动：至少 2 个产地、3 个“产地 × 批次”单元严格改善，且依次删除任一个构造批次单元后的平均成对效应仍大于 0。

若任一条未满足，仍如实报告锁定结果，但将等权集成降为“候选组合”或“敏感性分析”，并以锁定结果最优且组成最简洁的模型作为当前数据的最终预测器。不用较弱的基线、随机种子或事后子集替换该判定。

## 8. 必做消融、伪证与压力检验

1. **强基线**：raw PLS-DA、SNV-LR、SG1-收缩 LDA；
2. **集成消融**：单独报告 LDA、LR、温度校准 SVM 及三者等权集成，报告错误交集；
3. **选带和多视图负结果**：保留开发阶段未取得稳定增益的证据，不根据锁定结果重新启用；
4. **标签置换**：以构造批次为单位在开发集进行 200 次保持类别数的标签置换，检查管线泄漏或编号捷径；
5. **整幅来源图像迁移**：分别用全部 `*-1` 训练、`*-2` 测试，以及反向评估；其结果只作采集域压力检验；
6. **现代光谱深度基线**：在相同构造批次边界下评估 321,776 参数的三阶段 1D 残差 CNN。开发 OOF 对每个外折使用另一个构造批次早停，然后按所选 epoch 从头重训外折训练集；锁定测试将在全开发集上固定训练 88 epoch（开发折所选 epoch 的中位数取整），不得使用锁定标签早停或调参。该模型是现代对照，不是主方法候选。

## 9. 创新表述边界

已有研究已将高光谱、SG/SNV、PLS-DA/SVM、CNN、特征波长和多模型应用于酸枣仁或其他食品的产地鉴别。因此，本研究不把某个通用预处理、分类器、选带或软投票包装为首创模块。

当前数据下可辩护的方法学贡献是：面向八产地酸枣仁任务，把确定性构造批次、开发/锁定分离、折内温度校准、强单模型基线、严格成对不确定性、选带/多视图负结果、标签伪证和整幅图像压力检验整合为一个可审计的建模与验证流程。该贡献属于任务定制的验证与建模策略，而不是新的基础分类理论。
