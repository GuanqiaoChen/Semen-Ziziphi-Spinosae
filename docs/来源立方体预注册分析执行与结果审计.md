# 来源立方体预注册分析：执行与结果审计

## 1. 审计结论

本轮已在 `main` 分支上完成结果前冻结的方法、完整 GPU 执行、锁定统计后处理、逐预测复核与出版级图件质检。方法层面采用了当前 16 个来源立方体所允许的最严格设计；证据层面仍必须限定为**八个存档商业标签在两个配对来源立方体之间的闭集迁移审计**，不能改写为未见农场、独立批次、收获年份、供应商、设备、实验室或真实地理产地的外部认证。

最重要的实证结论不是“深度模型胜出”，而是：

1. 强 SNV–LR 基线的双向主要平衡准确率为 86.94%，明显高于两个神经模型；
2. 融合网络达到预声明的“有限空间排列支持”门槛，但其双向主要平衡准确率只有 45.94%；
3. 融合网络相对光谱神经消融仅高 1.19 个百分点，条件性区间跨越零；相对 SNV–LR 低 41.00 个百分点；
4. 因此，当前结果只支持“该融合模型在当前配对立方体迁移中使用了部分前景内部排列信息”，不支持“空间信息改善了最佳预测性能”，更不支持空间差异具有地理、组织学或化学因果解释。

## 4. 模型与主要估计量

- `snv_lr`：对每粒种子的前景均值光谱进行 SNV，再以开发侧内部验证选择正则化强度；3,144 个参数。
- `spectral_only`：一维残差光谱编码器；79,624 个可训练参数。
- `fusion_net`：与 `spectral_only` 共享光谱分支和分类头的初始参数，增加逐像素 392→16 投影与二维残差空间编码器；209,000 个可训练参数。
- 两个神经模型在相同方向、相同种子下的光谱分支和分类头初始化 SHA-256 已运行时核对一致。
- 主要预测器为三个预声明优化种子的温度校准概率平均；温度仅由开发侧内部验证数据确定。
- 主要估计量 θ 为两个迁移方向等权的 balanced accuracy；每个方向内八个标签/测试立方体等权。

## 5. 主要性能结果

| 模型 | 后缀 1→2 balanced accuracy | 后缀 2→1 balanced accuracy | θ | 条件性 95% 区间 |
|---|---:|---:|---:|---:|
| SNV–LR | 84.77% | 89.11% | **86.94%** | 75.38%–96.20% |
| 光谱神经消融 | 52.94% | 36.56% | **44.75%** | 25.31%–63.58% |
| 谱空融合网络 | 43.85% | 48.02% | **45.94%** | 31.98%–60.70% |

区间由固定随机种子 `20260721` 的 10,000 次八标签配对 block bootstrap 产生；全部模型和反事实共享同一重采样索引。它们只描述当前 16 个存档立方体条件下标签对贡献的不稳定性，不是新批次总体的抽样区间。

神经结果存在明显优化不稳定性：光谱神经消融在两个方向的 seed 级 balanced accuracy 分别为 12.50%–53.10% 与 12.34%–35.80%；融合网络分别为 30.19%–45.58% 与 41.86%–44.69%。第二方向的两个光谱神经校准温度还分别达到冻结网格的相反边界 0.25 和 4.0；结果后没有扩展温度网格。这些都是开发/优化诊断，不能作为独立重复，也进一步限制了神经概率的解释。

## 6. 空间机制证伪

预声明的主要机制效应为融合网络 `full − spatial_shuffle`：

- 后缀 1→2：+0.78 个百分点；
- 后缀 2→1：+25.52 个百分点；
- 两方向等权效应：**+13.15 个百分点**；
- 条件性 95% 区间：**+1.82 至 +24.38 个百分点**；
- 六个“方向 × 优化种子”配对差异均为正；
- 八标签对精确单侧 sign-flip：`p = 11/256 = 0.04296875`；
- 双侧敏感性值：`p = 22/256 = 0.0859375`。

该结果同时满足结果前写定的三个门槛：两个集成方向同为正、平均效应至少 2 个百分点、至少 5/6 个方向—种子差异为正。因此判定为**达到有限空间排列支持门槛**。但是两个方向的效应高度不对称，且双侧敏感性值高于 0.05，必须保留“方向依赖、当前立方体条件性证据”的限定。

其余预声明效应为：

| 效应 | 双向效应 | 条件性 95% 区间 |
|---|---:|---:|
| 融合 `spatial_shuffle − mean_broadcast` | +0.28 pp | −0.50 至 +0.99 pp |
| 融合 `full − mean_broadcast` | +13.43 pp | +1.71 至 +24.76 pp |
| 融合 `mean_broadcast − mask_only` | +11.99 pp | +4.10 至 +21.96 pp |
| 融合 `full − spectral_only(full)` | +1.19 pp | −11.29 至 +15.55 pp |
| 融合 `full − SNV–LR(full)` | −41.00 pp | −50.86 至 −31.46 pp |
| 融合 `mask_only − 12.5% chance` | +8.02 pp | −5.70 至 +23.54 pp |

`spatial_shuffle` 与 `mean_broadcast` 几乎相同，说明被打乱后的空间分支没有提供稳定的剩余优势。`mask_only` 点估计高于机会水平但区间跨越零，因此轮廓/裁切捷径仍是需要披露的替代解释，不能声称已被完全排除。

## 8. 图件结果后技术修正

首次后处理的数值表和统计检验一次通过，但人工目视发现方向标题把代码标识错误渲染为“1 → to → 2”，且混淆矩阵色条遮挡右侧面板。该问题只涉及版式，不涉及样本、概率、估计量、区间、检验或报告选择。

处理方式：

1. 首次目录完整移动为本地 `postprocessing_pre_figure_qa/`，未删除任何本地文件；
2. 增加方向标签单元测试并修正色条布局；
3. 从同一正式输入重新生成 canonical `postprocessing/`；
4. 逐文件 SHA-256 比较确认全部 CSV、JSON（除记录新图哈希的 manifest）及 `summary.md` 与首次运行一致，仅图件和 manifest 发生变化；
5. 修正后 15/15 单元测试通过，四组 PNG 均经人工目视复核。

这属于结果后、数值不变的出版图版式更正，不构成模型、效应、显著性或主张门槛的事后调整。

| 主张 | 当前判定 |
|---|---|
| 在当前配对来源立方体之间，SNV–LR 能较好区分八个存档商业标签 | 条件性支持 |
| 融合模型在当前迁移中使用了部分前景内部排列信息 | 有限、方向依赖地支持 |

当前版本可达到的核心贡献，是结果前冻结、实验层级诚实、强基线、同模型反事实、簇级条件性不确定性和完整可复核输出共同构成的方法学审计。确认性产地认证仍必须依赖《数据采集协作需求书》中规定的可追溯独立批次、跨年/跨设备外部测试与化学参照，而不是继续在这 16 个立方体上增加网络复杂度。
