# 当前数据条件下的顶刊级方法预注册

**冻结日期：** 2026-07-21

**状态：** 在任何新深度模型正式测试结果产生之前冻结

**适用分支：** `main`

**执行入口：** `deep_models/top_journal_current_data.py`

## 1. 研究定位与不可逾越的边界

本研究只评估八个存档商业样品标签在当前 16 个来源高光谱立方体之间的闭集迁移。每个标签只有两个来源立方体，文件夹后缀是否代表独立批次、供应商或采集会话尚无可核验元数据。因此，本轮方法升级可以提高模型开发、验证、证伪和报告的严谨度，但不能把当前数据升级为新农场、新批次、年份、仪器或实验室上的外部验证。

种子 patch 是来源立方体内的技术子样本，而不是 1,264 个独立产地重复。所有主要结论以来源立方体为评价边界；种子级结果只条件于当前 16 个立方体。

旧随机种子拆分和已经查看过的经典光谱结果均属于数据知情的探索性证据。以下尚未运行的新深度矩阵按本文件前瞻性固定，但仍不构成独立外部确认。

## 2. 研究问题与估计目标

### 2.1 唯一主要性能估计量

对模型或集成预测器 \(m\)，定义：

\[
\theta_m=\frac{1}{2}\left(BA_{1\rightarrow2,m}+BA_{2\rightarrow1,m}\right),
\]

其中 \(BA\) 为 balanced accuracy。因为每个测试方向恰有八个类别、每类对应一个来源立方体，\(\theta_m\) 等价于对 16 个测试来源立方体的类别召回率等权汇总。必须同时报告两个方向，不得只报告平均值或较好方向。

三个固定优化种子的**概率平均集成**是主要深度预测器；必须先逐样本平均八类概率，再计算一次集成指标。逐种子指标、均值、样本标准差和范围用于描述优化不稳定性，不能称为批次级置信区间。

### 2.2 唯一主要机制对比

主要机制对比为同一已锁定 `fusion_net` 在完整输入和前景内部空间打乱输入上的差异：

\[
\Delta_{spatial}=\theta_{fusion,full}-\theta_{fusion,spatial\_shuffle}.
\]

只有同时满足以下预声明门槛，才允许写成“当前配对来源立方体迁移中存在可用的前景内部空间排列信息”：

1. 概率集成的 `full - spatial_shuffle` 在两个方向均为正；
2. 双向平均差异至少为 0.02；
3. 六个方向×优化种子配对差异中至少五个为正；
4. 所有运行和反例均完整披露。

即使通过，也不得写成产地相关组织结构、空间因果机制或新批次泛化。

## 3. 数据冻结与拆分

- 固定读取 `data/0-1` 至 `data/7-2` 的全部同名 MAT/CSV 对，共 1,264 粒种子、392 个实测波长、16 个来源立方体。
- 主协议只有两个互为反向的来源隔离测试：全部 `*-1` 开发、全部 `*-2` 测试；以及全部 `*-2` 开发、全部 `*-1` 测试。
- 任一来源立方体不得同时出现在开发集和测试集。
- 测试集不得用于选模型、epoch、学习率、正则化、波段、随机种子、温度或输入变换。
- 每个开发方向内按每个来源立方体固定划分 80% 拟合、20% 验证。该验证只服务于训练停止、固定候选选择和温度缩放；因其仍来自同一开发立方体，必须称为“开发内种子级验证”，不得称独立或分组验证。
- 数据发现后记录每个 CSV 和 MAT 的内容 SHA-256、相对路径、文件大小、标签、来源立方体和后缀。

## 4. 固定模型矩阵

### 4.1 强光谱基线

- `snv_lr`：每粒种子的前景平均光谱做 SNV，再使用仅由开发集拟合的标准化和类别等权多项 logistic regression。
- 正则化候选 `C={0.01,0.1,1,10,100}` 仅由开发内验证选择；并列时按验证 NLL、较小复杂度和固定顺序裁决。
- 已执行的 SNV–LR、真实波长敏感 SG 一阶导数–LR、20 潜变量 PLS-DA 和 RBF-SVM 结果保留为外部于新深度代码的描述性强基线；不得因为新结果高低而删选。

### 4.2 新神经模型

- `spectral_net`：只使用前景平均光谱的 SNV 表示和一维残差光谱编码器，是与融合模型共享谱分支逻辑的深度光谱消融。
- `fusion_net`：组合一维残差光谱编码器与高效空间分支。空间分支先以逐像素 392→16 的可学习谱投影压缩波段，再使用二维残差编码；避免旧 3D 主干的巨大显存和不透明感受野成本。
- 两个神经模型使用相同的分类类别、开发/测试拆分、优化种子、停止规则和指标；不得为某一模型单独调参。

### 4.3 固定训练设置

- 优化种子：`42`、`2024`、`2025`；反事实置乱种子固定为 `9173`，与模型初始化种子分离。
- batch size 32；最多 60 epochs；至少 12 epochs；patience 8。
- AdamW，初始学习率 `1e-3`，weight decay `1e-4`；ReduceLROnPlateau 只读取开发内验证 NLL。
- 类别等权交叉熵、label smoothing 0.05、梯度范数裁剪 5.0；CUDA AMP 开启。
- checkpoint 选择唯一规则：验证 macro-F1 最大；若并列，验证 NLL 最小；若仍并列，选择最早 epoch。
- 温度缩放只使用开发内验证 logits，在固定 `[0.25,4.0]` 对数网格上最小化 NLL；同时保留未校准和校准预测。
- 不因任何测试指标增加 epoch、替换随机种子、改变架构或调整损失。

## 5. 同一模型的反事实输入

每个训练完成并锁定的模型不再拟合，直接在同一测试样本上成对评估：

1. `full`：完整前景高光谱 patch；
2. `spatial_shuffle`：只在前景内移动完整 392 波段像素光谱，保留轮廓和像素光谱分布，删除内部空间排列；
3. `mean_broadcast`：把该种子的前景平均光谱复制到其轮廓内，保留平均光谱与形态，删除像素异质性和内部排列；
4. `mask_only`：全部波段只保留二值轮廓，删除实测光谱与强度。

固定解释顺序：

- `full - spatial_shuffle`：内部空间排列；
- `spatial_shuffle - mean_broadcast`：无序像素异质性；
- `mean_broadcast - mask_only`：平均光谱；
- `mask_only - chance`：形态、居中、裁切或分割捷径。

对 `spectral_net`，空间打乱和均值广播理论上不应改变平均光谱；其预测一致性同时作为反事实实现的机器校验。

## 6. 结果、校准与不确定性

### 6.1 必须报告的指标

- accuracy、balanced accuracy、macro precision、macro recall、macro-F1；
- multiclass NLL、Brier score、固定 10 箱 ECE；
- 混淆矩阵、每类 recall、逐来源立方体 accuracy；
- 参数量、最佳 epoch、实际训练时长、温度和失败状态；
- 每粒种子的八类未校准与校准概率。

### 6.2 配对效应与集成

必须输出逐方向、逐模型、逐种子的成对反事实差异，以及三种子概率集成后的同一差异。次要对比包括：

- `fusion_net(full) - spectral_net(full)`；
- `full - mean_broadcast`；
- `mean_broadcast - mask_only`；
- 新模型相对同方向 SNV–LR、PLS-DA 和 SG 一阶导数–LR 的描述性差异。

不得把三个优化种子当作生物重复进行显著性检验。

### 6.3 条件性区间

- 对当前测试数据的比较采用来源立方体保持完整的成对重采样；模型和反事实条件共享相同重采样索引。
- 以八个标签对为 block 评估方向/标签异质性，并报告有限 block 数导致的不稳定性。
- 任何区间只条件于当前 16 个来源立方体，不得解释为新批次总体置信区间。
- 多个次要检验如被报告，使用 Holm 校正；主要机制对比只有一个。

## 7. 质量控制与完整交付

正式结果必须同时具备：

- 完整命令、开始/结束 UTC、运行时、Python/PyTorch/CUDA/h5py/GPU、Git commit 和锁文件；
- manifest、波长、数据内容哈希、拆分、训练历史、候选选择、逐样本概率、逐运行指标、逐立方体指标、集成指标、反事实效应和混淆矩阵；
- 自动测试证明来源立方体零交叉、概率和为 1、指标可由逐样本概率重算、全部数值有限、反事实保持预期信息；
- 不存在的输出不得以空表、占位图或预期值代替；中途失败不得写成完整执行。

## 8. 预声明判定规则

- 空间主张只按第 2.2 节门槛判断；不通过即明确报告“未获得稳定空间排列增益”。
- 若 `mask_only` 明显高于 12.5% 机会水平，必须把形态/裁切捷径列为主要替代解释；较低也不能反向证明形态完全无贡献。
- 融合模型只有在两个方向相对 `spectral_net` 一致改善时，才可描述为当前数据中的谱空增量；否则报告无稳定增益或方向依赖。
- 校准只描述当前配对立方体；没有未知类和独立校准集，不得声称可靠拒识或部署校准。
- 波段权重或敏感性只有在方向、优化种子和遮挡/移除分析中稳定时才能称模型敏感区间；没有配对化学测量时绝不称化学标志物。

## 9. 禁止主张

无论性能多高，本轮不得使用以下主张：地理产地认证、真实溯源或防伪系统；独立外部验证；新农场、批次、年份、供应商、设备或实验室泛化；化学成分或组织因果定位；Grad-CAM 单纳米解释；部署就绪；开放集识别；1,264 个独立生物重复。

本轮可争取的顶刊级贡献是：**来源立方体隔离、前瞻性固定、包含强基线、同模型反事实证伪、校准和层级不确定性的高光谱方法学审计案例**。数据证据不足时收窄结论，不降低方法标准。

## 10. 偏离管理

正式测试开始后，任何代码修复、硬件恢复或协议偏离都必须：

1. 在查看受影响的测试性能前停止；
2. 记录原因、影响范围和已暴露信息；
3. 使旧输出失效并保留审计副本；
4. 在中文修订总账新增更正记录；
5. 全矩阵按同一修复版本重跑，不得只重跑较差单元。
